# Contributing

Obrigado por contribuir com madbuilder-dicas! A documentação está organizada por idioma dentro da pasta `docs/`, atualmente apenas Português está disponível em `docs/pt/`.

Regras rápidas:

- Adicione novos artigos dentro de `docs/pt/` para Português.
- Use nomes de arquivo descritivos, em minúsculas, separados por hífen, ex: `uso-basico.md`.
- Se possível, mantenha traduções sincronizadas caso outras línguas sejam adicionadas futuramente.
- Abra uma issue para discutir mudanças grandes antes de implementá-las.

Processo técnico:

- Fork o repositório e crie uma branch com um nome descritivo.
- Faça commits pequenos e com mensagens claras.
- Abra um Pull Request para a branch `main`.

Obrigado! — Maintainers
